HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Within the head element, The title element defines the title for a webpage. This element is required and is extremely important for Search Engine Optimization (SEO)
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The section element organizes content on the webpage
  • h1 & h2 etc represents the headers on a webpage
  • ul creates an unordered list, and each line item is listened with li
  • ol creates an ordered list, and each line item is listened with li

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS is CSS put directly into the html file
  • External CSS sheets are linked into an HTML file

Git Commands

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone : clones a github repository onto the computer
  • git add -A: adds modifications in the current working branch to the new one
  • git commit -m “add starter code”: commit the files with a message describing what the commit contains
  • git pull origin main: check if the local branch is in sync with the base branch in github
  • git push origin branch-name: push branch to github

JavaScript

  • var keyword = variable, followed by listed variables in quotation marks
  • The console.log method outputs a message to the web console by adding an argument
  • We use conditional statements and loops to control the order in which code is executed
  • Conditional statements = if/else if statements
  • Array: single variable that is used to hold a group of data
  • For loops: performing a task on all items in an array by allowing a single code block to be executed over and over
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
